# Skill: Product Ideation & Requirement Engineering

## Description
This skill enables the AI agent to act as a Product Strategist. It can synthesize raw concepts into actionable product requirements, identify market/technical gaps, and refine user experience workflows.

## Capabilities
* **Concept Synthesis:** Transform high-level ideas into detailed value propositions.
* **Requirement Refinement:** Convert informal notes into structured **Product Requirement Documents (PRD)**.
* **Gap Analysis:** Identify edge cases, missing features, and technical risks in proposed ideas.
* **User Story Generation:** Create clear "As a... I want... So that..." stories with defined **Acceptance Criteria (AC)**.
* **Competitor Analysis:** Suggest pivots or unique selling points based on existing market solutions.

## Input Format
* **Raw Idea:** A brief description of the product or feature.
* **Target Audience:** The intended user base.
* **Constraints:** Budget, time, or technical limitations.

## Output Format
* **Executive Summary:** High-level vision.
* **Feature Roadmap:** Categorized list of MVP vs. Future features.
* **Requirement Table:** | Feature ID | Requirement | Priority | Rationale |
    | :--- | :--- | :--- | :--- |
    | F01 | [Description] | [P0-P3] | [Technical/Business value] |

## Best Practices
* Prioritize "Jobs to be Done" over specific UI elements.
* Always include "Non-functional Requirements" (Performance, Security, Scalability).
